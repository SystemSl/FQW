# Установка

<span>
pip install -r requirements.txt <br>
pip install torch torchvision --index-url https://download.pytorch.org/whl/cu118
</span>

# Скрипты

<span>
network.py — обучение сети <br>
evaluate_model.py — оценка модели на датасете KITTI <br>
evaluate_results.py — оценка результатов на датасете KITTI <br>
evaluate_results_sparse.py — оценка результатов на датасете KITTI при неполных данных <br>
process_video.py — оценка потока и диспаратности на стерео-видео <br>
process_video_2.py — оценка потока и диспаратности на стерео-видео с возможностью пропуска кадров <br>
video_maker.py — создание видео из кадров KITTI raw data <br>
mp4_to_gif.py — создание gif файла из mp4 <br>
test.py — оценка потока и диспаратности по 4 изображениям
</span>
